Soy Cristopher Miranda

Egresado de Ingeniería en Sistemas de la Universidad Evangélica Boliviana. Enfocado en desarrollo back-end y front-end con .NET (C#) y Angular. Experiencia en APIs RESTful con ASP.NET Core Web API utilizando Arquitectura Limpia, con bases de datos relacionales como SQL Server y MySQL, desplegados en Azure y Nginx. Competente en sistemas Linux (Ubuntu Server, Debian) y conceptos de redes (CCNA v7).
